🤖 WhatsApp Auto Reply (Namorada Edition)

Bot automatizado para responder mensagens no WhatsApp Web usando Playwright + IA (Gemini), simulando uma conversa natural e humana.

⚠️ Projeto para fins educacionais. Automação no WhatsApp pode violar termos de uso.

📌 Funcionalidades
📲 Lê mensagens automaticamente no WhatsApp Web
🧠 Gera respostas com IA (Google Gemini)
💬 Responde como se fosse uma conversa real
⏱ Simula tempo de resposta humano
🚫 Evita responder mensagens enviadas por você mesmo
🔁 Loop contínuo monitorando novas mensagens

🛠️ Tecnologias
Node.js
Playwright
Google Gemini API

Uso livre para estudos e projetos pessoais.
